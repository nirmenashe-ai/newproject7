## 🚀 Quick Start

### Prerequisites

- Python 3.8 or higher
- pip package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/interactive-dashboard.git
cd interactive-dashboard

# Install dependencies
pip install streamlit pandas plotly
```

### Run the Dashboard

```bash
streamlit run dashboard.py
```

The app will open automatically in your browser at `http://localhost:8501`.

## 📸 Dashboard Overview

### KPI Metrics

The top row displays four key performance indicators with trend deltas:

| Metric | Description |
|--------|-------------|
| **Total Sales** | Sum of monthly sales units |
| **Total Revenue** | Cumulative revenue in USD |
| **Total Users** | Total active user count |
| **Avg Engagement** | Mean engagement percentage |

### Interactive Chart

Select any metric from the sidebar to visualize its trend over a 6-month period (Jan–Jun) with an interactive Plotly line chart.
